8080 IDE

Project Overview

This project is an Integrated Development Environment (IDE) specifically designed for the Intel 8080 microprocessor. It features a custom assembler, real-time registers visualization, and a built-in emulator.

Developed with performance in mind, it provides developers with a seamless experience when writing and debugging legacy assembly code in a modern environment.

Key Features

  • Build in assembler and simulator.
  • Real-time syntax highlighting for 8080 Assembly.
  • Step-by-step instruction debugging.
  • Integrated memory viewer and hex editor.
  • Cross-platform compatibility.

Tech Stack

C++ Qt QMake
