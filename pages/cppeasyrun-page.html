CppEasyRun

Project Overview

CppEasyRun is a simple ide like program that is meant to make running short, single file C++ programs as easy as possible. It is designed for learning and demonstration purposes and not for developing applications. You don't even need to create file, it is created automaticly.

Key Features

  • LSP integration.
  • Syntax highligting.
  • Coding help.
  • Code diagnostics.

Tech Stack

C++ Qt CMake
