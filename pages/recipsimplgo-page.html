Project Overview

This project is a digital recipe book. It lets users save their recipes. It has a powerfull recipe creator witch lets users create recipes as html pages without knowing html.

Key Features

  • Easy to use recipe creator.
  • Modern and easy to use interface.
  • Marking recipes as favorite.

Tech Stack
